## 项目简介

本项目为计算机图形学课程实验，基于 **Python + Taichi** 实现二维粒子引力模拟。

程序通过计算多个粒子之间的万有引力，使粒子在二维平面内进行运动，并利用 Taichi GUI 实时渲染粒子位置，实现具有动态视觉效果的粒子系统。

---

## 实现功能

本项目主要完成了以下内容：

- 基于 Taichi 搭建二维粒子模拟程序
- 使用万有引力模型计算粒子之间的相互作用
- 实时更新粒子的位置与速度
- 使用 Taichi GUI 可视化粒子运动
- 支持 GPU（若不可用则自动切换至 CPU）运行

## 实现原理

本实验采用二维粒子系统进行模拟。

程序中的每一个粒子都会受到其它粒子的万有引力作用，根据牛顿第二定律计算粒子的加速度，再更新速度与位置。

每一帧执行流程如下：

1. 计算所有粒子之间的引力；
2. 更新粒子速度；
3. 更新粒子位置；
4. 清空画面；
5. 绘制所有粒子；
6. 显示下一帧。

Taichi 将这些计算并行化，因此能够高效完成大量粒子的实时模拟。

## 参数说明

程序中的主要参数位于 `config.py`：

| 参数 | 说明 |
|------|------|
| N | 粒子数量 |
| DT | 时间步长 |
| G | 万有引力常数 |
| SOFTENING | 防止粒子距离过小时数值发散 |
| PARTICLE_RADIUS | 粒子显示半径 |

这些参数均可自行修改，以观察不同模拟效果。
